¿Que es Freepto?

Freepto es un sistema GNU/Linux completo adapto para memorias USB. Esto significa que puedes llevar la memoria USB siempre con ti y utilizar cualquiera ordenador como si fuera tu portátil. Además los datos que guardarás dentro de la memoria será automáticamente cifrados (o sea no podrán ser leídos por otras personas).

La idea que está a la base del desarrollo de Freepto es la de ofrecer un sistema sencillo por la gestión segura de las herramientas mas utilizadas por los activistas, sin renunciar a la comodidad de un sistema operativo tradicional.

¿Come se usa?

Usa Freepto es muy sencillo.

Hace falta insertar la memoria USB en el ordenador cuando está apagado, luego encenderlo, seleccionar el boot desde USB, y Freepto se arrancará.

Para tener todas las informaciones su como instalar, arrancar utilizar Freepto, puedes leer la pagina de documentación.

Siempre con ti Un sistema operativo de bolsillo

Freepto es un sistema operativo completo dentro de una memoria usb. La puedes usar en cualquiera ordenador, y tendrá todo lo que necesitas. No se trata de un sistema para usar solo en casos de emergencia: Freepto es adapto para un uso habitual.

Cifrada pera custodiar los secretos

Todo el contenido de Freepto es cifrado con una contraseña que conoces sólo tu. De esta manera también si pierdes la llave, o te la roban, no tienes que preocuparte que tus datos se queden en manos de las personas equivocadas.

Todo incluido desde el correo hasta la gráfica

Freepto contiene muchos de los programas útiles: navegador, gestor de correos, editor de imágenes y si algo falta, lo puedes instalar gracias a synaptic, el gestor de paquetes presente también en Debian y Ubuntu.

Configurada para la seguridad

Hemos ententado de poner Freepto lo más segura posible sin que estos empeorase de alguna manera la facilidad de uso:

  • Los programas de chat y filezilla están configurados para el uso de tor, de manera que se tengan conexiones anónimas y seguras.
  • Firefox incluye las extensiones para cifrar la comunicació con los servidores el má posible.
  • Con Firefox es posible navegar hacia las paginas web .onion (paginas internas a la red tor de las cual no se conoce la posición).

Paranoia adicional opcional, pero a su alcance

Aumentar el propio nivel de seguridad debería ser opcional pero sencillo; por esto hemos incluido dentro de Freepto una serie de herramientas para quien quiere aumentarlo:

  • Cancelación segura de los ficheros
  • Truecrypt, para gestionar archivos cifrados
  • Torbrowser-launcher, para tener siempre la ultima versión de torbrowser y navegar de manera anonima
  • GPG, para intercambiarse correos cifrados
  • TorTP, para forzar el uso de TOR a todas las aplicaciones de red
  • Florence, para tener un teclado virtual donde insertar las contraseñas
  • Tomb, para la gestión avanzada de los archivos cifrados

Documentación incluida

Usar Freepto es sencillo, pero es siempre util tener la documentación para leer cualquiera necesidad. Por esto hemos incluido dentro Freepto su manual, que te ayudará a usarla de la mejor manera.

Hemos también incluido Crypt'r'die, un folleto dedicado a la defensa dal control digital que hemos producido: está organizado en sencillas "recetas" para mejorar la propia seguridad, echale un vistazo!

Personalízala

Si quieres, puedes crear una versión modificada de Freepto, incluyendo nuevos programas o haciendo modificas de cualquiera tipo. Freepto es basada en Debian Live Build, un conjunto de herramienta que permite de generar distribuciones live personalizadas basadas en Debian. PUedes contribuir empezando de nuestro repository.
