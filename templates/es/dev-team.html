Como colaborar

El desarrollo de Freepto se basa en el sistema Debian Live Build y en el desarrollo de aplicaciones para la seguridad de los usuarios.

El modelo de organización que seguimos es un modelo abierto y comunitario, así que puedes unirte al equipo de desarrolladores.

Por la comunicación del desarrollo se suele usar la lista de correo general o con un contacto directo entre los desarrolladores por cuestiones de menor importancia en nuestro canal IRC.

Las versiones de desarrollo la puedes encontrar en esta página: http://dev.freepto.mx/.

El repositorio

El código fuente es disponible al repositorio https://github.com/AvANa-BBS/freepto-lb/, de manera que el desarrollo quede publico y abierto también a quien quiera contribuir de manera esporádica.

El espacio de GitHub también es usado para gestionar los tickets y la timeline de desarrollo general.

Si tienes un bug que reportar puedes usar la página de GitHub o escribirnos un correo.

Crear un ambiente de desarrollo

Hemos creado un sistema para automatizar el proceso de instalación y configuración de un ambiente de desarrollo. El código se basa en las librerías fabric y fabtools de python.

Para mayor información puedes mirar el repositorio https://github.com/AvANa-BBS/freepto-buildtools.
