Cómo colaborar

El desarrollo de Freepto está basado en el sistema Debian Live Build y en el desarrollo de aplicaciones para la seguridad de los usuarios.

El modelo de organización que seguimos es un modelo abierto y colaborativo, así que estás invitado a participar.

Para la coordinación del proyecto usamos la lista de correo general, y para cuestiones de menor importancia o para un contacto directo entre desarrolladores usamos el canal IRC.

Puedes encontrar las versiones de desarrollo en esta página: http://dev.freepto.mx/.

El repositorio

El código fuente está disponible en el repositorio https://github.com/AvANa-BBS/freepto-lb/, de manera que el desarrollo es público y está abierto también a quien quiera contribuir de manera esporádica.

Usamos Git y el espacio de GitHub como sistema de control de versiones y también para gestionar los tickets y la línea de tiempo del desarrollo general.

Si tienes un bug que reportar puedes usar la página de GitHub o escribirnos un correo.

Un ambiente de desarrollo

Hemos creado una herramienta para automatizar el proceso de instalación y configuración de un ambiente de desarrollo. El código se basa en las librerías fabric y fabtools de python.

Para má información consulta el repositorio https://github.com/AvANa-BBS/freepto-buildtools.
