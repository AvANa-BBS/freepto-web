Descargar

Para verificar con seguirdad la intentidad de los servicios ofrecidos por Freepto, tienes que bajar e instalar la nuestra CA en tu ordenador.

wget http://ca.freepto.mx/ca.pem

Verificar

Es muy recomendado verificar la autenticidad del certificado descargado. Es posible hacerlo en simples pasos:

  • Baja nuestra clave GPG:

    • gpg --search-key 364B68D4

  • Verifica la huella digital de la clave:

    • Este paso es necesario para asegurarte que la clave gpg descargada sea efectivamente la nuestra

    Verifica que la huella digital que te muestra este comando,

    gpg --fingerprint 364B68D4

    es el mismo que está aquí:

    Key fingerprint = C355 97C8 7BE0 8D11 A5D8 4087 909C 6C75 364B 68D4

  • Verifica el certificato:

    • Descarga la signature del certificado:

    wget http://ca.freepto.mx/ca.pem.sig

    Ahora verifica el certificado usando gpg:

    $ gpg --verify ca.pem.sig ca.pem
    gpg: Signature made Mon 03 Nov 2014 23:45:25 GMT using RSA key ID 364B68D4
    gpg: Good signature from "Freepto Repository "

Instalación

Para instalar el certificado en tu ordenador, sigue estos pasos:

$ sudo -s
# apt-get install ca-certificates
# wget -O /usr/local/share/ca-certificates/freepto.pem http://ca.freepto.mx/ca.pem
# update-ca-certificates
