Scarica

Per verificare con sicurezza l'identità dei servizi offerti da Freepto, devi scaricare ed installare la nostra CA sul tuo computer.

wget http://ca.freepto.mx/ca.pem

Verifica

E' fortemente raccomandato verificare l'autenticità del certificato scaricato. E' possibile farlo in pochi semplici passaggi:

  • Scarica la nostra chiave GPG:

    • gpg --search-key 364B68D4

  • Verifica il fingerprint della chiave:

    • Questo passaggio e' necessario per assicurarti che la chiave gpg scaricata sia effettivamente la nostra

    Verifica che il fingerprint mostrato dal seguente comando,

    gpg --fingerprint 364B68D4

    sia uguale a quello mostrato di seguito:

    Key fingerprint = C355 97C8 7BE0 8D11 A5D8 4087 909C 6C75 364B 68D4

  • Verifica il certificato:

    • Scarica la signature del certificato:

    wget http://ca.freepto.mx/ca.pem.sig

    A questo punto verifica il certificato utilizzando gpg:

    $ gpg --verify ca.pem.sig ca.pem
    gpg: Signature made Mon 03 Nov 2014 23:45:25 GMT using RSA key ID 364B68D4
    gpg: Good signature from "Freepto Repository "

Installa

Per installare il certificato nel tuo computer, segui i seguenti passaggi:

$ sudo -s
# apt-get install ca-certificates
# wget -O /usr/local/share/ca-certificates/freepto.pem http://ca.freepto.mx/ca.pem
# update-ca-certificates
