Come collaborare

Lo sviluppo di Freepto si basa sul sistema Debian Live Build e nello sviluppo delle applicazioni per la sicurezza degli utenti.

Il modello di organizzazione che seguiamo è un modello aperto e comunitario, quindi se vuoi puoi unirti al gruppo di sviluppo.

La comunicazione dello sviluppo si svolge nella mailing-list generale o con un contatto diretto tra gli sviluppatori per questioni di minor importanza sul nostro canale IRC.

Le versioni di sviluppo le puoi trovare a questa pagina http://dev.freepto.mx/.

Il repository

Il codice sorgente è disponibile al repository https://github.com/AvANa-BBS/freepto-lb/, cosicché lo sviluppo sia pubblico e aperto a chi voglia partecipare anche in modo sporadico.

Lo spazio di GitHub è anche usato per gestire i ticket e la timeline dello sviluppo generale.

Se hai un bug da segnalare puoi usare la pagina di GitHub oppure scriverci una mail.

Creare un ambiente di sviluppo

Abbiamo creato un sistema per automatizzare il processo di installazione e configurazione di un ambiente di sviluppo. Il codice si basa su Vagrant

Per maggiori informazioni puoi guardare il repository https://github.com/AvANa-BBS/freepto-vagrant.
